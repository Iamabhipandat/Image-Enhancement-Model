# Gradio Demo: Image Equalization with Kornia

This repository contains a minimal interactive demo showcasing image
histogram equalization using **Kornia**, built with **Gradio** and deployed
on **Hugging Face Spaces**.

## Live Demo
[https://huggingface.co/spaces/Iamabhipanda/kornia-image-equalization](https://huggingface.co/spaces/Iamabhipandat/kornia-image-equalization)

## Overview
The demo allows users to:
- Upload an image
- Apply histogram equalization using Kornia
- Instantly visualize the equalized output

The goal is to provide an intuitive, hands-on example for understanding
image equalization in computer vision.

## Tech Stack
- Python
- Kornia
- Gradio
- Hugging Face Spaces

## Motivation
This project was created as part of a community contribution effort to
provide interactive examples and tutorials for Kornia users.
